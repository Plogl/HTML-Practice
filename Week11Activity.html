What is a Binary Search Algorithm?

From Wikipedia, A binary search tree, "In computer science, binary search, also known as half-interval search, logarithmic search, or binary chop, is a search algorithm that finds the positionof a target value within a sorted array. Binary search compares the target value to the middle element of the array. If they are not equal, the half in which the target cannot lie is eliminated and the search continues on the remaining half, again taking the middle element to compare to the target value, and repeating this until the target value is found. If the search ends with the remaining half being empty, the target is not in the array.

Binary Search Algorithm Process

  1. Compare your value with the middle element of a sorted array
  2. Check if the value matches. If it matches, the value is found in the list
  3. If the values do not match, compare the values to find if the value is greater or lesser than the element
  4. If the value is greater, search through the top half of the tree, and if lesser, the bottom half of the tree (excluding the original element)
  5. Continue this process until there is only one element remaining. If they are equal, the value is found. if not, the value does not exist in the list.
